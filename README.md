# eva01.vim

A dark color scheme for Vim/NeoVim

## Installation

If you are using [vim-plug](https://github.com/junegunn/vim-plug), add this to your `.vimrc` or `init.vim`

```vim
Plug 'hachy/eva01.vim', { 'branch': 'main' }
```

```vim
colorscheme eva01
" or
colorscheme eva01-LCL
```
