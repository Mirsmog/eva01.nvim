# eva01.vim

A dark color scheme for Vim

## Installation

If you use [vim-plug](https://github.com/junegunn/vim-plug)

```vim
Plug 'hachy/eva01.vim'
```

Add this to your .vimrc/init.vim

```vim
colorscheme eva01
" or
" colorscheme eva01-LCL
```
